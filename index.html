QSnapshot by legnaleurc

A screen capture tool

Download latest version (v0.1.0) for Windows Mac OS X

Feature

  • Capture fullscreen
  • Capture window under cursor
  • Capture rectangular region
  • Capture component of window

Preview

  • Select Rectangular Region

    You could resize or move after selection, to adjust region.

  • Select Window Component

    Furthermore, You could grab specific window component.
